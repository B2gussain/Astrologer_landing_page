Astrologer Landing Page

This is a landing page built with Next.js, TypeScript, and Tailwind CSS.
It was created as part of an internship assignment.

How to run

Clone this repo git clone https://github.com/B2gussain/Astrologer_landing_page.git 
cd Astrologer_landing_page

Install dependencies npm install

Start the dev server npm run dev

Open http://localhost:3000 in your browser

API routes

/api/hero → Hero section data
/api/testimonials → Testimonials data
Notes

Fully responsive (mobile, tablet, desktop)
Components are modular and easy to extend
